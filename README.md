# Обновление сайта IOSappTester

Эта папка содержит файлы, которые можно положить в корень репозитория GitHub Pages `IOSappTester`.

## Что заменить

- `index.html`
- `styles.css`
- `app.js`
- `manifest.json`
- `service-worker.js`

Папку `icons` можно оставить прежнюю, если она уже есть в репозитории.

## После загрузки

1. Открой репозиторий на GitHub.
2. Загрузи файлы из этой папки в корень репозитория.
3. Подтверди замену старых файлов.
4. Подожди 1-2 минуты, пока GitHub Pages обновится.
5. Открой `https://zhavokhiryunusov.github.io/IOSappTester/`.

Если на телефоне показывается старая версия, обнови страницу несколько раз или удали сайт с экрана "Домой" и добавь заново. Это связано с кэшем PWA.
