# ExpenseDepositApp

SwiftUI-приложение для iPhone: учет расходов, категории, статистика за месяц и калькулятор вклада со сложными процентами.

## Что внутри

- Локальное хранение через SwiftData, без сервера.
- Вкладки:
  - `Траты`: список расходов, добавление траты, удаление свайпом.
  - `Статистика`: сумма за выбранный месяц и разбивка по категориям.
  - `Вклад`: расчет сложных процентов с дневной или месячной капитализацией.
- Категории можно добавлять и удалять внутри приложения.
- Калькулятор вклада поддерживает пополнения:
  - без пополнений;
  - ежедневно;
  - каждые две недели;
  - ежемесячно.
- После расчета показывается отчет по месяцам: пополнения, проценты и итоговый баланс.

## Как запустить

1. Откройте `ExpenseDepositApp.xcodeproj` в Xcode на Mac.
2. В настройках target `ExpenseDepositApp` при необходимости поменяйте `Bundle Identifier` с `com.example.ExpenseDepositApp` на свой.
3. Выберите iPhone Simulator или подключенный iPhone.
4. Нажмите Run.

Минимальная версия: iOS 17.0, потому что приложение использует SwiftData.

## Проверка сборки через GitHub Actions

В проект добавлен workflow `.github/workflows/ios-build.yml`.

Как использовать:

1. Создайте новый репозиторий на GitHub.
2. Загрузите содержимое папки `ExpenseDepositApp` в корень репозитория.
3. Откройте вкладку `Actions`.
4. Запустите workflow `iOS Build` вручную через `Run workflow` или сделайте push в `main`.

Этот workflow проверяет сборку для iPhone Simulator на macOS runner. Для установки на реальный iPhone или сборки `.ipa` отдельно нужны Apple signing certificates/profiles.
