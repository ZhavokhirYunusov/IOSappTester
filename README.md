# ExpensePWA

PWA-версия приложения для учета расходов и расчета вклада. Работает бесплатно через браузер и может добавляться на экран iPhone как отдельное приложение.

## Что есть

- расходы с категориями;
- добавление и удаление трат;
- статистика за месяц;
- локальное хранение в браузере через `localStorage`;
- калькулятор вклада со сложными процентами;
- дневная и ежемесячная капитализация;
- пополнения ежедневно, каждые 2 недели, ежемесячно или без пополнений;
- отчет по месяцам;
- `manifest.json` и `service-worker.js` для PWA.

## Как загрузить на GitHub Pages

1. Создайте или очистите репозиторий на GitHub.
2. Загрузите содержимое папки `ExpensePWA` в корень репозитория.
3. Откройте `Settings` -> `Pages`.
4. Выберите `Deploy from a branch`.
5. Branch: `main`, folder: `/root`.
6. Нажмите `Save`.
7. Откройте ссылку GitHub Pages на iPhone в Safari.
8. Нажмите `Поделиться` -> `На экран "Домой"`.

После этого приложение появится на главном экране iPhone.
